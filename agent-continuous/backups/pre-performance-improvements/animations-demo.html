How to Add Videos to Your Landing Pages

1. Prepare Your Videos

  • Export videos in MP4 (H.264) and WebM formats for broad compatibility
  • Keep files under 5MB for fast loading (use compression tools)
  • Recommended dimensions: 1920x1080 for full-width, 1280x720 for cards
  • Frame rate: 30fps is sufficient for web
  • Create seamless loops (end frame matches start frame)

2. Add Videos to Your Project

<video class="apple-video"
       data-autoplay="true"
       data-loop="true"
       muted
       playsinline>
  <source src="assets/videos/demo.mp4" type="video/mp4">
  <source src="assets/videos/demo.webm" type="video/webm">
</video>

3. For Canvas Image Sequences

  • Export your animation as a sequence (After Effects, Blender, etc.)
  • Name files: frame-0001.jpg, frame-0002.jpg, etc.
  • Optimize images (compress to ~50-100KB each)
  • Place in assets/images/sequence/

4. Alternative: Use GIFs or Animated SVGs

For simpler animations, optimized GIFs or animated SVGs can provide similar effects with smaller file sizes and easier implementation.
