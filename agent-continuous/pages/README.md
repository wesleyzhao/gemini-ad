# Gemini AI Landing Pages Collection

12 stunning, Apple-inspired landing pages for Gemini AI advertising campaigns.

## Pages Overview

1. **valentine.html** - Valentine's Day love letter theme with hearts and romantic infographics
2. **writers.html** - For writers with VO3 voice features, elegant typography focus
3. **creators.html** - For video creators with Nano Banana, visual/creative showcase
4. **operators.html** - Google Workspace integration with sliding panel demo
5. **automators.html** - Automation workflows with visual flow diagrams
6. **apple-style.html** - Pure Apple.com minimalist aesthetic with bundling concept
7. **trust.html** - Citations and fact-checking emphasis, academic feel
8. **research.html** - Academic/professional researchers, paper-writing focus
9. **productivity.html** - Time-saving features, efficiency metrics
10. **workspace.html** - Deep Google Workspace integration showcase
11. **comparison.html** - Gemini vs competitors feature table
12. **future.html** - Aspirational/premium future-of-work theme

## Features

All pages include:
- Standalone HTML with complete styling
- Apple.com-inspired minimalist design
- Mobile responsive (works on all devices)
- Smooth animations using shared animations.js
- Clear CTAs and hero sections
- Distinct personality and messaging
- Links to shared CSS (/workspace/gemini-ads-agent/project/assets/css/shared-styles.css)
- Links to shared JS (/workspace/gemini-ads-agent/project/assets/js/animations.js)

## Usage

Each page is GitHub Pages compatible and can be:
- Deployed individually
- Used in ad campaigns
- A/B tested
- Customized for specific audiences

## Design Philosophy

- **Minimalist elegance** with generous whitespace
- **Premium, aspirational feel**
- **Smooth animations and transitions**
- **Product-centric with clear hierarchy**
- **Mobile-first responsive design**
- **Short attention span optimization**

All pages are optimized and under 25KB each for fast loading.
