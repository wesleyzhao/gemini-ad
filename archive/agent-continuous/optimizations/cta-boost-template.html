üéØ Best Practices

  • Single Primary CTA: Use enhanced style for main action only
  • Secondary CTAs: Use secondary style to avoid competing
  • Copy: Action-oriented (Start, Try, Get vs Learn, Discover)
  • Clarity: Add subtext to reduce friction ("No credit card required")
  • Contrast: Ensure CTA stands out from page background
  • Mobile: Ensure minimum 44px touch target (we use 56px)
  • Accessibility: Maintain sufficient color contrast (4.5:1+)
