üìö Implementation Guide

1. Include Required Files

<link rel="stylesheet" href="assets/css/cta-variants.css"> <script src="assets/js/ab-testing.js"></script> <script src="assets/js/cta-copy-variants.js"></script>

2. Create A/B Test Variants

<!-- Variant A --> <a href="#" class="cta cta-primary" data-ab-test="my-test" data-ab-variant="A" data-ab-copy="Get Started Free"> Get Started Free </a> <!-- Variant B --> <a href="#" class="cta cta-secondary cta-arrow" data-ab-test="my-test" data-ab-variant="B" data-ab-copy="Try Gemini Now"> Try Gemini Now </a>

3. Track Conversions

// When user completes desired action (signup, purchase, etc.) ABTest.trackConversion('my-test');

4. View Results

// In browser console ABTest.displayResults('my-test'); // Or get results programmatically const results = ABTest.getResults('my-test'); console.log(results);
Pro Tip: The framework automatically assigns users to variants, tracks clicks, and stores data in localStorage. No server required!
