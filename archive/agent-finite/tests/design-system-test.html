Responsive Testing

To test responsive behavior, resize your browser window or use DevTools device emulation:

  • Desktop: 1440px+ (full typography scale)
  • Tablet: 735px - 1068px (scaled down typography, 2-col grids)
  • Mobile: < 735px (smallest typography, 1-col grids)

Accessibility Features

The design system includes:

  • Keyboard focus indicators (try tabbing through links/buttons)
  • Reduced motion support (respects prefers-reduced-motion)
  • High contrast mode support (prefers-contrast: high)
  • Dark mode support (prefers-color-scheme: dark)
  • Screen reader utilities (.sr-only class)
  • Semantic HTML structure
